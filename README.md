# Book_Tracker
Приложение для учёта прочитанных книг с GUI на tkinter.
Автор: Юльмухаметов Айдар.

## Возможности
* Добавление книг с указанием названия, автора, жанра и количества страниц.
* Фильтрация по жанру и количеству страниц.
* Сохранение данных в JSON-файл.

## Использование
1. Запустите `BookTracker.py`.
2. Добавляйте книги через форму ввода.
3. Используйте фильтры для поиска.
4. Сохраняйте данные в JSON.

## Требования
* Python 3.6+
* Стандартные библиотеки: tkinter, json, os, subprocess
